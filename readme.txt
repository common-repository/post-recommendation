Post Recommendation show next post to visitors (blog readers) in recommended box. It helps visitors to know about next post. The recommended box can be placed in any one of three positions Top right, Bottom left and Bottom right. You can show post image and description in recommended box and these are optional.  

== Description ==

Post Recommendation Home Page - http://www.apptha.com/category/extension/Wordpress/post-recommendation

*	Show the next post or Post in light popup box.
*	Allow the user to select the default image for Post.
*	Set  popup box position at bottom-left , bottom-right and top-right.
*	facebook and  twitter  like buttons for every  Post.
*	Set popup box content  by select  Post Image and Description  at back end.


== Installation ==

= Minimal Requirements =
* PHP: 5.x.x
* mySQL: 4.0, 4.1 or 5.x
* WordPress: 2.8 or newer

= Recommended Requirements =
* PHP: 5.2.x or newer
* mySQL: 5.x
* WordPress: 2.8 or newer

= Basic Installation =
1. Download the current version of the Post Recommendation plugin go to Plugins --> click on add new --> upload -->select 
   post_recommendation.zip --> install.
2. Activate the Post Recommendation through the 'Plugins' menu in WordPress
4. In admin menu PICA Gallery menu is created that's it! You're done.


== Other Notes ==

= To change the global settings of the Post Recommendation =

4. Select the popup window postion
5. upload default image for post.
6. You have give the Facebook and twitter buttons to share.

== Frequently Asked Questions ==
= What is this extension for? =
By using Post Recommendation, you can show next blog or post to visitors (blog readers) in recommended box. It helps visitors to know about next post. The recommended box can be placed in any one of three positions Top right, Bottom left and Bottom right.
You can show post image and description in recommended box. 

= What is the list of versions it is compatible with? =
It is compatible with Wordpress upto 3.3.1
